Задание 1.
Разработать класс «Учреждение образования», который должен содержать следующие
элементы:
статическое поле министр образования, поле вид учреждения образования
(университет, колледж, школа и т.д.), поле год основания, поле название,
статический конструктор, в котором статическое поле принимает значение Журавков,
конструктор с параметрами, конструктор без параметров, который вызывает
конструктор с параметрами и , свойства для полей класса,
обеспечивающие полный доступ
(для года основания проверять, чтобы год был не меньше 1900), деструктор,
который выводит сообщение об уничтожении объекта, метод ToString() 
для строкового представления объекта.
 
В методе Main нужно создать объект класса «Учреждение образования», причем при
вызове конструктора порядок аргументов не должен соответствовать порядку
параметров в сигнатуре конструктора (см. именованные параметры).
Год основания вводить с клавиатуры, а остальные поля задавать в виде констант.
Вывести информацию об объекте. Изменить фамилию министра образования на Косинов.
Написать фрагмент кода, который позволит увидеть работу деструктора.

===============================================================================

Задание 2.
Разработать класс «Книга», который должен содержать следующие элементы:
закрытое поле название, закрытое поле автор, свойства (только для чтения)
этих полей класса; статическое поле издательство,
свойство для доступа к этому полю;  автоматические свойства тираж, цена книги;
конструктор с необязательными параметрами; деструктор, который выводит сообщение
«Уничтожена книга … »;   статический конструктор, метод  возвращающий стоимость
всего тиража; статический метод, который принимает в качестве параметров два
объекта класса, а в качестве результата возвращает общий тираж этих книг.

В методе Main нужно создать два объекта класса «Книга» с издательством «РОСМЭН»,
заданным через статический конструктор. Данные для первой книги задавать в
виде констант, а для второй вводить с клавиатуры. Вывести информацию о книгах,
включая стоимость издания и общий тираж. Заменить издательство на «Михалыч®».
Написать фрагмент кода, который демонстрирует работу деструктора.

===============================================================================

Задание 3.
Разработать класс «Университет», который должен содержать следующие элементы:
поле ректор, поле количество студентов, поле год основания, поле название;
конструктор с параметрами, конструктор без параметров,
который вызывает конструктор с параметрами
(в качестве его аргументов использует Тимошин, 1000, 1968, ГГТУ им.П.О. Сухого);
свойства для полей класса, обеспечивающие полный доступ
(для числовых проверять, чтобы год был не меньше 1900); метод ToString() для
строкового представления объекта; статический метод для сравнения двух 
университетов, который принимает в качестве параметров объекты класса,
а в качестве результата возвращает -100, если количество студентов в первом
университете меньше второго, ноль, если количество студентов одинаковое  и 100,
если в первом университете учится больше студентов.

В методе Main нужно создать два объекта класса «Университет».
Первый создавать с использованием конструктора без параметров, а второй,
используя конструктор с параметрами, причем при вызове конструктора порядок
аргументов не должен соответствовать порядку параметров в сигнатуре
конструктора (см. именованные параметры).
Поля задавать в виде констант.
Вывести информацию об объектах. Определить, какой из университетов
обучает больше студентов.
